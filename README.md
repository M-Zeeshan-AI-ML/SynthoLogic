# SynthoLogic
Privacy-First AI Data Generator by Structural Mind.

Project Overview:
A high-fidelity tabular data synthesizer built with Python. It utilizes Gaussian Copulas to model multivariate distributions and preserve complex correlations between features.

Key Features:

Automated PII Masking: Uses Faker and Regex to detect and replace sensitive identifiers.

Statistical Fidelity: Employs empirical quantile transformations to ensure synthetic data matches the original marginal distributions.

Privacy Audit: Generates a real-time 'Privacy Score' to assess the risk of data leakage.

Streamlit UI: A clean, production-ready interface for non-technical users.
